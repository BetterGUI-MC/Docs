# {{ $frontmatter.title }}

This action will execute the command as a player

## Format

* `<command>`
* `player: <command>`

## Note

* This is the default type of all actions, so you can specific the action without the prefix `player:`

## Example

* `spawn`
* `player: warp pvp`
* `player: me Hello World!`
